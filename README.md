# gpt-research-in-mining
Исследование возможностей чат-ботов с искусственным интеллектом для решения задач добывающей промышленности
## Автор: Михаил Ситулин

## Цель исследования
Сравнить качество решения прикладных задач горнорудной отрасли с помощью зарубежного и отечественного ИИ-ассистентов.

## Используемые модели
1. Зарубежный бот: Gemini.
2. Отечественный бот: Алиса AI.

## Выбранные кейсы
1. Диагностика насосного оборудования: Моделирование ситуации с падением давления и ростом вибрации в насосе центробежного типа.
2. Промышленная безопасность: Алгоритм действий при задымлении на руднике.

## Основные выводы
* Зарубежный бот: Проявил себя лучше в объяснении физики процессов, но давал более общие инструкции.
* Отечественный бот: Дал более подробные ответы на поставленные кейсы и более полно отнес ответы к теме вопросов.

## Методология
1. Сформулирован единый запрос для каждого кейса.
2. Запросы отправлены обоим ботам в одном временном контексте (новый чат).
3. Результаты сохранены в папку `results/`.
4. Проведен сравнительный анализ по критериям: полнота ответа, соответствие
отраслевой специфике, наличие "галлюцинаций" (ложных фактов), удобство восприятия.
